# Systematic Landscape Scan: AI-Assisted Development
**Research Date**: January 2026  
**Method**: GitHub API systematic search across 10 dimensions

---

## Executive Summary

The AI-assisted coding landscape is **larger and more mature** than initially assessed. Key findings:

1. **Spec-Driven Development is exploding** - 6+ major frameworks with 80k+ combined stars
2. **MCP is the integration standard** - 50+ significant MCP servers, 35k+ combined stars
3. **Multi-agent orchestration maturing** - ChatDev (28k), Claude-Flow (11k)
4. **Workflow platforms dominating** - n8n (166k), Langflow (143k), Dify (124k)
5. **Context management critical** - Multiple approaches emerging

**Market Size**: 80k+ combined stars across 6+ frameworks

**Convergent Patterns**:
- Constitution/standards first
- Requirements → Design → Tasks → Implement
- User stories as independent slices
- Approval gates between phases

**Divergent Approaches**:
| Approach | Projects | Philosophy |
|----------|----------|------------|
| **Autonomous** | Spec Kit | Specs generate code |
| **Collaborative** | OpenSpec, cc-sdd | Human-AI alignment loops |

## Key Patterns Identified

### 1. The SDD Standard Workflow

```
Constitution → Specify → Plan → Tasks → Implement → Archive
     ↓            ↓         ↓       ↓         ↓          ↓
  Standards    WHAT      HOW    Breakdown  Execute    Merge
```

All major SDD tools follow this or a subset.

### 2. The 15+ AI Tool Problem

Every major framework now supports 10-15 AI tools:
- Claude Code, Cursor, Windsurf, VS Code
- Gemini CLI, Codex, Qwen, Auggie
- GitHub Copilot, OpenCode, etc.

**Implication**: Tool-agnostic is table stakes.

### 3. The Dashboard Trend

Rising pattern: Real-time web dashboards for workflow visibility
- spec-workflow-mcp: VSCode extension + web dashboard
- Spec Kitty: Kanban dashboard
- Task Master: Progress tracking

### 4. The MCP Convergence

MCP is becoming the standard for:
- Tool integration
- Memory persistence
- Agent communication
- Workflow orchestration

## Conclusion

The landscape is **significantly larger** than initially assessed:

1. **SDD is mainstream** - 80k+ stars across frameworks
2. **MCP is the glue** - Universal integration layer
3. **Dashboards are rising** - Visual feedback expected
4. **Multi-tool support is table stakes** - 15+ tools per framework
5. **Brownfield matters** - Not just greenfield 0→1

**Lia's opportunity**: Focus on transparency and education while ensuring MCP compatibility. The market is crowded at the "autonomous execution" end; there's space at the "guided learning" end.
